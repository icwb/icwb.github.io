DISCLAIMER


ICWB hanya bertujuan untuk hiburan semata. Botmin hanya bermaksud untuk membuat sesuatu seperti WorldWarBot 2020 dan CivilWarBot 2020 dengan tema Indonesia. Apapun yang diposting oleh bot hanyalah simulasi. Jadi mohon jangan terlalu serius menanggapi apapun yang diposting oleh bot ini.

Botmin tidak bertanggungjawab jika ada oknum yang memanfaatkan materi dari page ini, atau oknum yang terinspirasi dari page ini untuk membuat segala macam bentuk kerusuhan yang dapat memecah belah kesatuan NKRI.

Link untuk membuat template meme breaking news: https://breakyourownnews.com/
  • Apa itu Indonesian Civil War Bot 2030/ICWB?

    ICWB adalah page bot sejenis WorldWarBot 2020 dimana bot ini otomatis memposting simulasi invasi dari suatu daerah ke daerah lain secara acak.

  • Apakah ICWB beneran bot?

    Postingan yang tidak diawali dengan [Botmin post] kemungkinan besar diposting oleh bot. Botmin juga bisa memposting tanpa diawali dengan [Botmin post].

  • Apa itu Botmin?

    Admin Page/Developer Bot/yang buat bot/yang punya ICWB.

  • Ini game bukan sih min? Cara mainnya gimana?

    Ini bukan game yang bisa kalian mainkan. Dan sepertinya page ini gak bisa disebut game.

  • Bagaimana sih cara kerja bot ini? Apakah cara kerjanya sama kayak WWBot 2020?

    • Daerah pertama dipilih secara acak
    • Daerah kedua dipilih secara acak dari daerah yang berdekatan dengan daerah pertama
    • Penyerang ditentukan dari pemilik daerah pertama
    • Lawan ditentukan dari pemilik daerah kedua
    • Penyerang menguasai daerah kedua dari Lawan
    • Ada kondisi dimana penyerang kemungkinan gagal menguasai daerah kedua
    • Kondisi diulangi sampai satu provinsi tersisa

    Bisa dibandingkan dengan WWBot 2020

  • Apakah project botnya open-sourced?

    Iya. Bisa dilihat di Github ICWB. ICWB menggunakan bahasa pemrograman Typescript/Javascript (NodeJS). Boleh diclone, boleh dimodifikasi jika ingin membuat page seperti ini.

  • "Min kan [Nama Provinsi] menguasai daerah lain, tapi kok daerahnya sendiri dikuasai sama provinsi lain?"

    Suatu provinsi masih dapat menguasai daerah manapun jika provinsi tersebut masih menguasai setidaknya satu daerah.

  • "[Nama provinsi] nerf dong min", "Min awas kalo [Nama Provinsi] sampe kalah", dan lain-lain.

    Postingan tidak boleh dimanipulasi oleh pihak ketiga. Ada pengecualian untuk botmin hanya jika terjadi sesuatu seperti bug atau error dari program.

  • "Bikin versi War antar kota/kabupatennya min"

    Tidak, tidak akan pernah.

  • "Bikin grup dong min"

    Gua pikir-pikir dulu.

  • "Min kok Season 2 nya lama banget"

    Mohon maaf karena delaynya hampir 3 tahun.

    Sebelumnya gua berpikir untuk tidak melakukan Season 2 karena memang sudah tidak relevan. Tapi karena gua lihat WWBot memposting run ke-4 nya. Gua berpikir untuk melakukan Season 2 juga.

  • "Min katanya mau nambahin blablabla..."

    Tidak jadi.

  • "Bikin yang kayak gini gimana min? Cara buat bot gimana min?"

    Bikin sesuatu seperti page ini tidak selalu membutuhkan bot, tetapi akan lebih melelahkan dan terkesan "biased" jika dilakukan secara manual. Kalian boleh mengambil inspirasi dari cara kerja bot ini dari source code-nya, dan membuat bot-mu sendiri.

    Untuk membuat bot, setidaknya kalian harus punya pengalaman membuat sebuah program. Kalian bisa liat tutorial untuk membuat bot di website Bot Appreciation Society Wiki
